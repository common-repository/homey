== Description ==
Share your Homey Flows on your WordPress site.

Simply paste the Flow's URL and the Flow will show up on your website.

== Installation ==
1. Upload the "homey" folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
